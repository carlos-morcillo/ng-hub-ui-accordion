## Resumen

`ng-hub-ui-accordion` es una biblioteca de acordeones para Angular 21 centrada en accesibilidad, composición declarativa y personalización mediante variables CSS.

## Instalación

```bash
npm install ng-hub-ui-accordion
```

## Uso rápido

```typescript
import { AccordionComponent, AccordionPanelComponent } from 'ng-hub-ui-accordion';
```

Consulta la documentación de Hub UI para la guía completa de instalación, API y ejemplos interactivos.

## Características

- Compatible con Angular standalone.
- Soporte de accesibilidad y navegación por teclado.
- Composición por paneles y plantillas personalizadas.
- Estilado mediante variables CSS.
